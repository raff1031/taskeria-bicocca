## Deploy su GitHub Pages

1. Vai su **Settings → Pages**
2. Source: `Deploy from a branch`
3. Branch: `main` · Folder: `/ (root)`
4. Salva — il sito sarà online su `https://[username].github.io/[repo-name]`

Il file `index.html` è completamente self-contained (nessuna dipendenza esterna necessaria).
